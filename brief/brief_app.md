Brief Aplikasi

Judul
Dianuin

Deskripsi singkat
Dianuin membantu kamu mengubah format file media. Kamu dapat melakukan konversi video, audio, dan gambar dalam satu aplikasi.

Fitur utama
• Konversi video ke format lain seperti MP4 ke MKV atau MOV ke MP4
• Ekstrak audio dari video seperti MP4 ke MP3
• Konversi gambar melalui perintah ffmpeg
• Kompresi ukuran video
• Ubah resolusi dan kualitas video sesuai kebutuhan
• Konversi otomatis berdasarkan preset yang kamu pilih

Target pengguna
• Konten kreator
• Mahasiswa multimedia
• Pengguna yang sering olah media di HP atau laptop

Nilai tambah untuk pengguna
• Satu aplikasi untuk semua kebutuhan konversi media
• Format output fleksibel
• Proses cepat menggunakan ffmpeg

Teknologi
• Flutter sebagai UI
• GetX untuk manajemen state, route, dan dependency
• ffmpeg_kit_flutter_new untuk pemrosesan media

Struktur arsitektur
• View berfokus pada tampilan
• Controller menangani logika konversi dan interaksi
• Binding untuk menyambungkan controller
• Service untuk menjalankan perintah ffmpeg

Jika kamu ingin, saya bantu buat struktur folder dan tampilan awal halaman konversi dengan GetX.
